\section{Модификация ИС ОПСАПК на основе автоматизированного выбора модели источника ошибок.} \label{sect3_2}
Рассмотрим ИС ОПСАПК, база источников ошибок в которой содержит только модели из класса скрытых полумарковских моделей. Расширим БУИМ из ИМ ЦПК (см. рис. \ref{Deundyak-fig1}) модулем подбора математической модели источника ошибок. Этот модуль получает на вход поток ошибок, зарегистрированный в реальном канале связи. Далее на основе метода подбора, предложенного в разделе 3, модуль выбирает из базы моделей источников ошибок, находящейся в БИВВ, ту модель, которая генерирует потоки ошибок, наиболее близкие к полученному.
Расширенный таким образом БУИМ будем обозначать БУИМ(А).

Выбранная модулем подбора математической модели источника ошибок модель подается блоком БУИМ на блок БИВВ, который осуществяет генерацию потоков ошибок с помощью этой модели в рамках имитационных экспериментов.

Модель источника ошибок может выбираться как из всего множества базовых моделей, так и из некоторого его подмножества, выбранного на основе каких-либо дополнительных соображений.
